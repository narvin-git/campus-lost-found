# Campus Lost & Found System

A full-stack web application built using:

- Node.js
- Express.js
- MySQL
- HTML, CSS, JavaScript

## Features
- Submit lost/found reports
- View all reports
- Search and filter
- Update status
- Delete reports
- Dashboard statistics
- 404 page handling
- Performance optimization (Lighthouse 100)
